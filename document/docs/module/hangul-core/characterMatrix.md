## characterMatrix

초/중/종성을 분리한다.

```ts
characterMatrix('안녕하세요! Hello~')
// [
//   [ "ㅇ", "ㅏ", "ㄴ" ],
//   [ "ㄴ", "ㅕ", "ㅇ" ],
//   [ "ㅎ", "ㅏ" ],
//   [ "ㅅ", "ㅔ" ],
//   [ "ㅇ", "ㅛ" ],
//   [ "!" ],
//   [ " " ],
//   [ "H" ],
//   [ "e" ],
//   [ "l" ],
//   [ "l" ],
//   [ "o" ],
//   [ "~" ]
// ]
```
