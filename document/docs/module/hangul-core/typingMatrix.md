# hangul-core

## typingMatrix

초/중/종성을 타이핑 순서로 분리한다.

```ts
typingMatrix('안녕하세요! Hello~')
// [
//   [ "ㅇ", "아", "안" ],
//   [ "ㄴ", "녀", "녕" ],
//   [ "ㅎ", "하" ],
//   [ "ㅅ", "세" ],
//   [ "ㅇ", "요" ],
//   [ "!" ],
//   [ " " ],
//   [ "H" ],
//   [ "e" ],
//   [ "l" ],
//   [ "l" ],
//   [ "o" ],
//   [ "~" ]
// ]
```
