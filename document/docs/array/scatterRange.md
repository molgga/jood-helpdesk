# scatterRange

source 배열의 분포도 정보를 반환합니다.

```ts
const result = scatterRange({
  source: [130, 128, 131, 134, 142, 150, 149, 147, 143],
  min: 100,
  max: 200,
  split: 11
});
```

